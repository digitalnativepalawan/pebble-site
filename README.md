# Pebble Site Builder

A no-code website builder for hospitality businesses — resorts, restaurants, cafes, and more.

## Getting Started

1. Clone this repo
2. Set up a new Supabase project and run the migrations in `/supabase/migrations`
3. Copy `.env.example` to `.env` and add your Supabase credentials
4. Run `npm install && npm run dev`
5. Visit `/admin` to set up your site

## Features

- Block-based page editor (Text, Image, Video, FAQ, Table, Stats, Timeline, Columns, Calculator, Form)
- Media library with Supabase storage
- Multi-page support
- Site settings (name, logo, colors, contact info)
- Drag & drop block reordering
- Mobile-first design

## Deploy

Deploy to Vercel or Netlify. Add your Supabase environment variables in the dashboard.

Built with React + Vite + Tailwind + Supabase.
